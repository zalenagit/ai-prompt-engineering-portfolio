# Before & After Examples

## Example 1 — Summarization
Input: Long paragraph about travel.
Output: A short 3-sentence summary capturing purpose, key points, and conclusion.

## Example 2 — Tone Transformation
Input: "Send me the report now."
Output: "Could you please share the report when you have a moment?"

## Example 3 — JSON Extraction
Input: "John met Sarah in Dallas on March 3."
Output:
{
  "person1": "John",
  "person2": "Sarah",
  "location": "Dallas",
  "date": "March 3"
}
